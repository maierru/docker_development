Add '127.0.0.1 *.localhost' to your /etc/hosts to make available http://grafana.localhost/ (and others) or use with port (for example, http://localhost:3000)

# FAQ Pro-Tip: To keep browsers from treating .localhost (and .test) domains as search terms, include the protocol when typing it into the URL field. For example: http://grafana.localhost rather than grafana.localhost
